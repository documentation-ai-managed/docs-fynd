## Recent Updates

Stay current with Fynd's evolution. You receive automatic notifications for major releases through your dashboard at `https://dashboard.example.com`. Review changes below to understand impacts on your integration.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "bugfix"]}>

## New Features

- Introduce real-time analytics dashboard for monitoring API usage and performance metrics in real time.
- Add webhook support for event-driven notifications, including `user.signup` and `payment.failed` events.

## Improvements

- Optimize query performance by `>30%` for high-volume endpoints.
- Enhance error messages with detailed debugging information.

## Bug Fixes

- Resolve intermittent authentication failures during peak hours.
- Fix pagination issues in list endpoints returning incomplete results.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "improvement"]}>

## New Features

- Launch multi-factor authentication (MFA) integration with TOTP providers.
- Enable custom domain mapping for your branded dashboard.

## Improvements

- Streamline SDK initialization with automatic token refresh.
- Update documentation with interactive API playground.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["breaking", "feature"]}>

## Breaking Changes

- Remove deprecated `legacy` authentication endpoint; migrate to OAuth 2.0.
- Change default rate limit from `{1000}` to `{500}` requests per minute.

## New Features

- Core API launch with user management, payments, and analytics endpoints.
- SDKs for JavaScript, Python, and Go.

## Initial Setup

- Bootstrap your integration with the new client libraries.

</Update>

## Upgrading to the Latest Version

Follow these steps to update your Fynd integration seamlessly.

<Steps>
  <Step title="Check Compatibility" icon="check-circle">
    Review breaking changes in the release notes above.
  </Step>
  <Step title="Update SDK" icon="download">
    <CodeGroup tabs="JavaScript,Python">
````javascript
npm install @fynd/sdk@latest
````
````python
pip install fynd-sdk==1.2.0
````
    </CodeGroup>
  </Step>
  <Step title="Test Endpoints" icon="play">
    Run your test suite against `https://api.example.com/v1`.
  </Step>
  <Step title="Deploy" icon="rocket">
    Monitor for errors using the new real-time dashboard.
  </Step>
</Steps>
