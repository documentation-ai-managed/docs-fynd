## Overview

To use Fynd Trail APIs, you need an API key linked to your account.

## Steps to Get Access

1. Login at `https://partners.fynd.com`
2. Navigate to **Trail Documentation**
3. Click **My API Keys**
4. Select **Generate New Key**
5. Copy your key and keep it secure

## Using Your API Key

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" https://trail.fynd.com/api/docs
```
